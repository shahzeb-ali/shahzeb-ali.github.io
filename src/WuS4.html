def fitLine(x_observed, y_observed):
    '''
    Fits a line f(x) = m * x + b to the observations
    
    Arguments:
        x_observed -- x values of the data
        y_observed -- y values of the data
    Returns:
        m          -- slope
        b          -- intersect
    '''
    mytmp=np.cov(m=x_observed,y=y_observed)
    a = mytmp[0,1]/x_observed.var()
    b = y_observed.mean() - a*x_observed.mean()
    return a,b

Gewicht / Größe n) [8 Punkte]¶

Vergleichen Sie Ihre Implementierung mit der Ausgabe der Funktion scipy.stats.pearsonr. Was bedeutet der zweite Wert den die Funktion scipy.stats.pearsonr zurückgibt?

Bei Correlation(x,y) erhält man Korrelationskoeffizent genau ein wert zurück wo bei pearsonr funktion ergibt 2 werte zurück nämlich Korrelationskoeffizent und p-value. Beide Funktionen geben fast gleiche(correlation gerundet) Korrelationskoeffizent zurück. Der p-Wert für jede unabhängige Variable testet die Nullhypothese, dass die Variable keine Korrelation mit der abhängigen Variable hat. Wenn es keine Korrelation gibt, gibt es keinen Zusammenhang zwischen den Änderungen in der unabhängigen Variable und den Verschiebungen in der abhängigen Variable. Die Nullhypothese ist, dass die beiden Variablen unkorreliert sind. Der p-Wert ist eine Zahl zwischen Null und eins, die die Wahrscheinlichkeit angibt, dass Ihre Daten entstanden wären, wenn die Nullhypothese wahr wäre.
